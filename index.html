先生だけ：メール/パスワードでログイン（Firebase Auth）

※Authの「承認済みドメイン」に GitHub Pages ドメインを追加してね（Firebase Console → Authentication）。
① 課（Lesson）を追加
保存先：lessons/<grade>/<lessonId>
② 単語を登録（CSV）
1行：word,meaning,wrong1,wrong2[,wrong3,wrong4,wrong5]
保存先：words/<grade>/<lessonId>/<stage>/<autoId>
③ 学生名簿 / 進捗（一覧）
学生：名前→匿名ログインで UID は内部だけに使う（画面には出さない）
名前 最終アクセス 概要
ユーザー：users/<uid> / 進捗：progress/<uid> / 完了：stageComplete/<uid>
重要（最初にやる設定）
1) Firebase Console → Authentication → Sign-in method → 匿名 と メール/パスワード を有効化
2) Authentication → Settings → Authorized domains に github.io（またはあなたのPagesドメイン）を追加
3) Realtime Database → Rules を下の「推奨ルール」にする（このindex想定）

推奨ルール（コピペ用） 
{
  "rules": {
    ".read": "auth != null",
    "users": {
      "$uid": {
        ".write": "auth != null && auth.uid === $uid"
      }
    },
    "progress": {
      "$uid": {
        ".write": "auth != null && auth.uid === $uid"
      }
    },
    "stageComplete": {
      "$uid": {
        ".write": "auth != null && auth.uid === $uid"
      }
    },
    "lessons": {
      ".write": "auth != null && auth.token.email != null" 
    },
    "words": {
      ".write": "auth != null && auth.token.email != null"
    }
  }
}
先生（メールログイン）だけが words/lessons を編集できます。
学生（匿名）は自分の users/progress/stageComplete だけ書けます。
